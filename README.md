# tinker_hack_4

React + TypeScript project bootstrapped with Vite.

## Tech Stack

- React 19
- TypeScript 5
- Vite 7
- ESLint 9

## Prerequisites

- Node.js 20+ (LTS recommended)
- npm 10+

## Setup (for teammates)

```bash
git clone <your-repo-url>
cd tinker_hack_4
npm install
npm run dev
```

App runs locally at the Vite URL shown in terminal (usually `http://localhost:5173`).

## Available Scripts

- `npm run dev` – start development server
- `npm run build` – type-check and create production build
- `npm run preview` – preview production build locally
- `npm run lint` – run ESLint

## Project Structure

```text
src/
  App.tsx
  main.tsx
  index.css
  App.css
public/
```
